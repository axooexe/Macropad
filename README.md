# ⌨️ Macropad
A sleek, fully custom 3x3 macro pad built from the ground up for developers and creators. Designed with a custom PCB, a tailored 3D-printed enclosure, and snappy firmware mapped to a hyper-efficient creative workflow.

# Project Breakdown

The Hardware & PCB
Brain: kb2040 dev board

Hardware Controls: 8 Hot-swappable MX mechanical switches + 1 Rotary Encoder for smooth volume scrolling.

Power: kb2040's usb C port

The Enclosure

# Key Features
Zone-Mapped 3x3 Layout: A grid intentionally split into three workflows: Dev Sandbox on the left, System/Media in the middle, and Design/Comms on the right.

Zero Hotkey Conflicts: Uses a global GUI + ALT hyper-modifier matrix. It launches your apps instantly without messing up your standard app shortcuts like Ctrl+C.

Tactile Hardware Controls: 8 mechanical keys paired with a dedicated rotary encoder for smooth, analog volume scrolling right at your fingertips.

Custom PCB Architecture: Fully custom-routed board with hot-swappable sockets to swap switches without soldering, running on a modern Type-C interface.

Ergonomic, Snug Enclosure: 3D-printed case designed in Blender with tight, zero-rattle tolerances and a low-profile angle to keep your wrists happy.

Lag-Free Firmware: Lightweight matrix scanning with built-in debounce handling and zero ghosting for instant input response.
