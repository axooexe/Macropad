A sleek, fully custom 3x3 macro pad built from the ground up for developers and creators. Designed with a custom PCB, a tailored 3D-printed enclosure, and snappy firmware mapped to a hyper-efficient creative workflow.

# Technical Specifications

Hardware:
Microcontroller: Adafruit KB2040 (RP2040) via USB-C

Keys: 8 MX-compatible mechanical switches (Hot-swappable)

Extra Controls: 1 Rotary encoder (for volume/scrolling)

Software & CAD Design:
PCB Design: KiCad

Enclosure Modeling: Blender, FreeCAD...

Firmware Development: VS Code
